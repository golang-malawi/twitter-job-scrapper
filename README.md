# Twitter-Job-Scapper
A telegram Bot that searches for jobs posts on twitter based on phrases. 

## Used Packages
- [telegram-bot-api](https://github.com/go-telegram-bot-api/telegram-bot-api)
- [twitter-scrapper](https://github.com/n0madic/twitter-scraper)
